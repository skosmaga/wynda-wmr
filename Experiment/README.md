# Experiment

This folder contains MATLAB scripts and datasets for **Wheeled Mobile Robot (WMR) parameter estimation** using experimental data.

## Files

- **`wynda_wmr_exp.m`**  
  WyNDA-based WMR parameter estimation using experimental data.

- **`sin_wmr_exp.m`**  
  SINDy-based WMR parameter estimation using experimental data, used for comparison.

- **`wynda_wmr_dt.m`**  
  Digital twin comparison between parameter estimation results obtained from a 3D model and WyNDA.

- **`data_exp/`**  
  Experimental datasets used for WMR parameter estimation.

- **`data_dt/`**  
  Experimental datasets used for digital twin simulation.

## Notes

These files support the experimental validation and digital twin analysis presented in the associated journal paper.
