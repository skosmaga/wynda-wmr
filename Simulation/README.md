# Simulation

This folder contains MATLAB scripts for **Wheeled Mobile Robot (WMR) parameter estimation** using simulation data.

## Files

- **`wynda_wmr_sim.m`**  
  WyNDA-based WMR parameter estimation using simulated data.

- **`sin_wmr_sim.m`**  
  SINDy-based WMR parameter estimation using simulated data, used for comparison.

- **`wynda_wmr_sim_change.m`**  
  WyNDA-based WMR parameter estimation with piecewise time-varying parameters.

## Notes

All scripts are used to support simulation studies presented in the associated journal paper.
